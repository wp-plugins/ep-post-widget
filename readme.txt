== Description ==

Add a "latest posts" or "latest [your_category] news" post widget to you sidebar or other widget area with ease. Not a million different settings, but still customizable!

Want it to suit your site better? Just edit the style.css file in the plugins folder to your liking.

== Installation ==

1. Upload the zipped file to yoursite/wp-content/plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
